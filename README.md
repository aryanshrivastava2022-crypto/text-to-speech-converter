# Text-to-Speech Converter
A simple Python application that converts text into speech using the `pyttsx3` library.

# Features
- Convert text into speech
- Offline text-to-speech conversion
- Easy command-line interface
- Supports different voices (depending on the operating system)

# Requirements
- Python 3.x
- pyttsx3

# Installation
Install the required library:
pip install pyttsx3

# How to Run
1. Download the project files.
2. Open a terminal in the project folder.
3. Run:
text_to_speech.py
4. Enter any text.
5. The application will read the text aloud.
